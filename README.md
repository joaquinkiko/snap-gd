# SnapGd

Snapshot based multiplayer synchronization for [Godot 4](https://godotengine.org/)

## Features

- Time synchronization between players
- Client predication and Server reconciliation for seemless player experience
- Interpolation for smooth synchronization regardless of network conditions
- Delta-compression for bandwidth management
- Lag compensation for fast-paced gameplay

## Install

After adding the latest Release to your project, enable the addon in your project settings

Note: this is designed for [Godot 4.6+](https://godotengine.org/download/archive/4.6-stable/), and is not guranteed to work for previous versions
